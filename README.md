# musicclient
> Play music from your favorite streaming services. All in one place. 

Use one music client to play music from multiple streaming services.

## Building

Windows:
```bash
git clone https://github.com/philliphqs/musicclient.git
cd musicclient
npm i
yarn app:dist
```
